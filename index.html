ℹ️ How It Works

            
ALEEM AI uses a secure backend proxy for all AI requests

            

              
1
Your message is sent to a Netlify serverless function — never directly to Google.

              
2
The function securely adds the Gemini API key (stored as a server environment variable) and forwards the request.

              
3
The AI response is returned to your browser. Your API key is never visible to anyone.

              
4
Automatic model fallback is built in — if one model is busy, the next is tried automatically.

            

            

              ⚠️ Free tier limits (2026): Gemini 2.5 Flash — 10 RPM, 250 RPD. Flash-Lite — 15 RPM, 1000 RPD.
